自分のやり方。ほかにもできる方法はある。Anacondaがいいって見るときあるな、よく知らない

1. Pythonをパソコンにインストールする。最新版なら多分大丈夫

2. IDLEを開く。Windowsなら画面左下のWindowsをクリックしてPythonかIDLEかで検索。デスクトップかツールバーにもってきとくといい

3. 白い正方形の画面が出てくる。Shellというらしい。>>>のところに「print("3+5=" + str(3+5))」と打ってみよう。「3+5=8」と出てくるはず。

File --> Open... で出てくる画面にフォルダを作る。名前は適当

フォルダ内に下の9ファイルを作る。打つの面倒なのでコピペをうまく使ってください：

supplement_1-3_light.txt

下のURLすべてにアクセスする。txtファイルは上で作ったファイルに中身をコピーする。行数が30万を超えるものもあるのでドラッグではなく「すべて選択」で。 StarsShow.pyはたぶん勝手にダウンロードされるので、上のフォルダ内に移す。 GitHubを見れるならこのPlaNavフォルダ内のReadPlease.txt以外のすべてのファイルをコピーでもOK。

StarsShow.pyを開く

ライブラリをインストールする。たぶんここが難関。「Python ライブラリ　インストール」とか調べてみるとやり方が分かると思う（以下でWindowsの場合を書く）。
プログラム冒頭でmatplotlib、numpy、tkinter、math、datetime、time、linecache、requests、jsonとたくさんのライブラリをインポートしているうち、どれがインストールしないとだめでどれがしなくて使えるか忘れてしまった、ごめんなさい
少なくともmatplotlib、numpy、tkinterは必要

Windowsの場合:
Windowsマーク
「cmd」と検索
コマンドプロンプトが開く
「pip install matplotlib」と入力、Enter
何かいろいろ出てきて勝手に終わる。
最後にsuccessfully installedか何か書いてれば成功。
ありませんみたいなのでるとスペルミスかインストール不要かのどちらかかな多分
matplotlibの部分をtkinterやnumpyに替えてほかもインストールする

StarsShow.pyに戻り、上のRun --> Run Module で実行する。WindowsならF5でもできる インストールしていないライブラリがあるとShellにmatplotlib is not foundのようなエラーが出るので、怖がらずにコマンドプロンプトでインストールする。

エラーなくうまくいけば準備は終わり。楽しんでいただけることを願っています。
